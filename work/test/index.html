1.2 商品

  1. 活动商品

    仅在活动页显示，与正常商品不共享库存，无法在正常列表页看见

2 主线场景

2.1 正常流程

2.2 新用户注册后购买

  1. 点击banner
  2. 查看列表，点击购买
  3. 弹出登陆注册页
  4. 注册成功
  5. 弹出订单页
  6. 支付成功
  7. 分享

2.3 新用户登陆后购买

  1. 点击banner
  2. 查看列表，点击购买
  3. 弹出登陆页
  4. 登陆成功
  5. 弹出订单页
  6. 支付成功
  7. 分享

2.4 老用户登陆后不能购买

  1. 点击banner
  2. 查看列表，点击购买
  3. 弹出登陆页
  4. 登陆成功
  5. 弹出老用户不能下单
